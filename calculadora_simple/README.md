# Calculadora Simple

Una aplicación de consola en Python que permite realizar operaciones aritméticas básicas de manera interactiva. El usuario puede elegir entre suma, resta, multiplicación y división, e ingresar los números a operar. El programa valida las entradas y maneja errores comunes, como la división por cero o la introducción de datos no numéricos. El ciclo se repite hasta que el usuario escribe `salir`.

## Requisitos del sistema

- Python 3.10 o superior
- No requiere librerías externas

## Instalación

1. Clona o descarga este repositorio en tu equipo.
2. Asegúrate de tener Python instalado (versión 3.10+).

## Ejecución

Desde la terminal, navega a la carpeta del proyecto y ejecuta:

```bash
python calculadora.py
```

## Ejemplo de uso

```
🧮 CALCULADORA SIMPLE
==================================================
Operaciones disponibles:
  • suma
  • resta
  • multiplicacion
  • division
  • salir (para terminar)
==================================================

🔢 Ingresa los números para la operación: suma
Primer número: 10
Segundo número: 5

------------------------------
```
